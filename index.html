Introduction


Lensless imaging allows us to design ultra-thin, lightweight cameras by replacing the lens with a thin optical mask. Due to the absence of a lens, the sensor in a lensless camera records a multiplexed version of the scene, and a computational algorithm is needed to invert this multiplexing. Due to their ultra-thin and lightweight nature, lensless cameras have potential applications in head-mounted displays (HMD), robotics, and medical imaging. In this project, we explore one crucial application of lensless cameras - stereo imaging.

Inversion using Trainable Inversion Layer

The Wiener Deconvolution operation described above can be implemented as a layer in an end-to-end stereo-matching network with the parameters of the Point Spread Function learned from the training data.
Calibration of lensless cameras to obtain the Point Spread Function (PSF) can be a time-consuming process and has to be done for each individual camera. Even a tiny error in calibration can lead to severe degradation in the performance of the reconstruction algorithm. Using a trainable inversion layer obviates the need for careful calibration.

Lensless Stereo Network


Stereo Matching algorithms require abundant texture information in the image to perform well. High-frequency information is lost in Wiener-reconstructed images because of an inherent tradeoff between noise and blur. In addition, the inversion process is highly sensitive to noise owing to the large support of the lensless PSF, and the regularization parameter needs to be chosen carefully.
Using a trainable inversion layer on the left and right measurements mitigates the above issues. Furthermore, training the inversion layer end-to-end with the stereo network allows the inversion layer to output features most beneficial for the specific task at hand.
